# parametrikusG
This program is a Lagrange, Bézier, and Catmull-Rom spline editor.
This was my second homework for the Graphics course at BME.

## Zoom
- 'Z' - Zoom out
- 'z' - Zoom in

## Pan
- 'P' - Pan right
- 'p' - Pan left

## Modes
- 'l' - Lagrange
- 'b' - Bézier
- 'c' - Catmull-Rom

## Catmull-Rom tension
- 'T' - grow the tension parameter
- 't' - shrink the tension parameter
